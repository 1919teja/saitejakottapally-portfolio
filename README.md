
# Saiteja's Terminal Portfolio

A high-performance, SRE-themed portfolio built with React + Vite + Tailwind CSS.

## üöÄ Deployment Guide

### 1. Initialize Git
If you haven't already, run these in your project folder:
```bash
git init
git add .
git commit -m "initial commit: terminal_interface_v2"
```

### 2. Connect to GitHub
Create a new repository on GitHub named `your-username.github.io` (for your main site) or `portfolio` (for a sub-page).
```bash
git remote add origin https://github.com/your-username/your-repo-name.git
git branch -M main
git push -u origin main
```

### 3. Enable GitHub Pages
1. Go to your repo on GitHub.
2. Click **Settings** > **Pages**.
3. Under **Build and deployment** > **Source**, select **GitHub Actions**.

The provided workflow in `.github/workflows/deploy.yml` will now handle everything automatically every time you push!
